Jobify Application

This was a project I started back in December 2022.The aim of this project was like job search application. Where Admin can create Job for users and track all records & status of that particular Job.

The backend API is made using Node.Js & Express. For the authentication i have use JWT authentication.

I have used MongoDB to store data, as it is comfortable to use for highly complex data structure..

Only Admin has ability to modify job, create job and control over user.No any user can modify or touch internal structure.

For the UI I have used React.Js and Chart.Js for beautiful representation of Data. Once user hover on Bar chart the Data will appear like tooltip animation.

Technologies:
  • - ReactJS
  • - MongoDB
  • - Node.js
  • - JWT
  • - Redux
