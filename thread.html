Thread Clone App

This was a project I started back in December 2023 to improve my Next.Js skill. I was the sole creator/developer.

The system is designed same as thread, where user can create communities, create thread and reply to other users.

A user can create the account and can write in thread by their choice.They can join any communities and reply any of the thread.

User can update , delete their own thread.

I worked on this system for one month before deploying it.

Technologies:
  • - Next.Js
  • - MongoDB
  • - ShadCn
